# プロトタイプ確認項目

## 子ども画面
- [ ] 大人の説明がなくても、最初の操作を理解できる
- [ ] 誤反応後に画面を閉じず、再挑戦またはヒントを選べる
- [ ] 「今日はここまで」を自分で選べる
- [ ] 振り返りを正解当てではなく、自分の感覚として選べる
- [ ] ごほうび演出が漢字の観察より目立ちすぎない

## 教材
- [ ] 1つの誤字形につき違いが1点だけである
- [ ] 正しい字形を最後に必ず再提示している
- [ ] 一画・部品ターゲットが学習上妥当である
- [ ] 選択肢に複数の正解が生じていない
- [ ] 画順選択肢の位置が判別可能である

## 記録
- [ ] ヒント利用を失敗として表示しない
- [ ] 途中終了でも発見が保存される
- [ ] 固定的な「○○タイプ」と表示しない
- [ ] 記録の書き出しと削除が動作する
