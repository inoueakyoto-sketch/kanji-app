# かんじたんけん iPad正式運用版

CodeSandboxのViteプロジェクトとして起動し、共有iPadのSafari／ホーム画面Webアプリで使用する構成です。

## 起動

```bash
npm install
npm run dev
```

## 正式運用機能

- 匿名プロフィールごとの学習記録・設定
- 起動時の利用者確認
- おとな用PINと自動ロック
- プロフィール追加・編集・保管・削除
- 現在／全利用者のJSONバックアップ
- 追加読み込み・全体入れ替え・直前読み込みの取り消し
- CodeSandbox、iPad Safari、PWA対応
